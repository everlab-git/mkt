# Freya CMS (monorepo)

Este repositório contém o **Editor** (React/Vite) e a **API** (Node/TypeScript) do Freya CMS.

## Estrutura

```
apps/
  editor/   # React + Vite + TS
  api/      # Node + TS (Hono)
```

## Requisitos

- Node.js 20+
- pnpm 9+

## Como rodar local

```bash
pnpm install
pnpm dev
```

## Variáveis de ambiente

Copie `.env.example` para `.env` e preencha.
